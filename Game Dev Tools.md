We need to be able to edit our world yaml, characters, dialog, scenes, and story beats. Basically we need to manage the game content.

## World Builder
This is a game content authoring tool. It should have the ability to communicate with the Grimoire Engine itself as it is basically a frontend editing tool for it. We want to be able to author entire sections of content via LLMs.

This is an estimated list of features. We want to be able to save and load a current world. 
* Dialog tree editor - view, search, add/remove/edit nodes. 
* Character editor - Includes a field that will allow call to llm to help generate details based on a prompt. Templated. Include a way to view, search, add/remove/edit. Ability to visualize and edit affiliations.
* Story Beat editor - graph of story beats with their contained scenes, linked to other scenes with conditionals
    * Scene editor
* See and edit world state
